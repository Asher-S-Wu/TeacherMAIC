# 课堂生成流程

## 前提

- 使用网站：https://teacher.li-xue.com/
- 模型服务商已在 Vercel 环境变量或页面设置中配置。
- 如果启用语音、图片、视频、联网搜索或 PDF 高级解析，对应服务商也需要配置。

## 页面流程

1. 打开个人网址。
2. 输入想学习或讲授的主题。
3. 可选：上传 PDF 资料。
4. 可选：开启联网搜索、图片生成、视频生成、语音合成等能力。
5. 点击生成课堂。
6. 等待课堂内容生成完成后进入课堂页面。

## API 边界

- 课堂生成接口仍使用项目现有 API 路径，不因品牌替换而改名。
- 不修改数据库名称、访问码 cookie、课堂 ID 或导出格式。
- 出现服务商认证、模型、Base URL 错误时，应修正服务端配置后再重试。

## 输出要求

返回课堂链接时使用当前个人网址域名：

```text
https://teacher.li-xue.com/classroom/{id}
```
