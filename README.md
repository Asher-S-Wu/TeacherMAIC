# 李雪 AI 教育平台

北京市朝阳区白家庄小学李雪老师的个人 AI 教育平台。

This project powers Li Xue's personal AI education platform for Beijing Chaoyang District Baijiazhuang Primary School. It turns lesson topics, teaching materials, or PDFs into interactive learning classrooms.

## Project Info

- Site name: 李雪 AI 教育平台
- Organization: 北京市朝阳区白家庄小学
- Owner: 李雪
- Website: https://teacher.li-xue.com/
- Deployment: Vercel Pro

## Features

- Generate lessons from text prompts or PDF materials.
- Create slides, whiteboard content, quizzes, and classroom structure.
- Support AI teacher narration, discussion, and Q&A.
- Support interactive learning scenes such as simulations, mind maps, games, and coding activities.
- Export PPTX and classroom resource packages.

## Usage

Visit https://teacher.li-xue.com/ and enter the topic or material you want to teach or learn.

Model, voice, image, video, web search, and PDF parsing capabilities depend on provider configuration in Vercel environment variables or the in-app settings page. Third-party provider names such as OpenAI, Gemini, MiniMax, MinerU, and Vercel are kept unchanged so the real service source remains clear.
