# 李雪 AI 教育平台

北京市朝阳区白家庄小学李雪老师的个人 AI 教育平台。

平台用于把课程主题、教学资料或 PDF 文档生成互动式学习课堂，帮助学生通过讲解、幻灯片、随堂测验、互动模拟和项目活动完成学习。

## 项目信息

- 网站名称：李雪 AI 教育平台
- 所属机构：北京市朝阳区白家庄小学
- 负责人：李雪
- 个人网址：https://teacher.li-xue.com/
- 部署方式：Vercel Pro

## 主要功能

- 课程生成：输入学习主题或上传资料，生成课堂内容。
- AI 讲解：支持 AI 教师讲解、课堂讨论和问答。
- 教学课件：自动生成幻灯片、白板内容和课堂结构。
- 随堂测验：生成题目并给出学习反馈。
- 互动学习：支持模拟实验、思维导图、小游戏、在线编程等互动内容。
- 资源导出：支持导出 PPTX 和课堂资源包。

## 使用说明

访问 https://teacher.li-xue.com/ 后，按页面提示输入想学习的内容，或上传 PDF 文件生成课程。

模型、语音、图片、视频、联网搜索等能力需要在 Vercel 环境变量或页面设置中配置对应服务商的 API Key。第三方服务名称如 OpenAI、Gemini、MiniMax、MinerU、Vercel 等保持原名，方便识别真实服务来源。
